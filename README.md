# Rugby Timer App for Garmin Connect IQ

## Overview

The Rugby Timer is a comprehensive Garmin Connect IQ application designed for rugby enthusiasts, coaches, and referees. It provides a dedicated platform to accurately track game time, manage scores, monitor yellow and red cards, and handle various game states (conversions, penalties, kickoffs, halftime). This app ensures you stay on top of the game, whether it's a 7s or 15s match, with essential timing and scoring features right on your wrist.

## Features

*   **Game Time Tracking:** Accurate tracking of match duration, with support for customizable half lengths (7s or 15s).
*   **Score Management:** Easily record and adjust scores for home and away teams, including tries, conversions, penalties, and drop goals.
*   **Card Management:** Implement yellow and red card timers with distinct durations for 7s and 15s rugby.
*   **Special Timers:** Dedicated countdowns for conversions, penalty kicks, and kickoffs.
*   **Game State Awareness:** Visual indicators for various game states: Playing, Paused, Conversion, Penalty, Kickoff, Halftime, and Game Ended.
*   **Session Persistence:** Automatically saves game state, allowing you to resume a match even after navigating away or restarting your device.
*   **Event Logging:** Tracks significant game events for review.
*   **GPS Tracking (Optional):** Records distance and speed during activity for later analysis.
*   **Vibration Alerts:** Notifies users with vibrations for critical timer events (e.g., 30 seconds remaining, card expiry).
*   **UI Lock:** Prevents accidental button presses during intense gameplay.

## How to Use

1.  **Start a New Game:** From the idle screen, press the 'Select' button to choose between Rugby 7s or Rugby 15s.
2.  **Start/Pause/Resume Game:** Use the 'Select' button to toggle between playing and paused states.
3.  **Record Scores:** Use the 'Previous Page' button to access the scoring menu (Try, Conversion, Penalty, Drop Goal).
4.  **Record Cards:** Use the 'Next Page' button to access the card management menu (Yellow Card, Red Card).
5.  **Adjust Scores/Undo Events:** Access the main menu (usually 'Menu' button) for options like adjusting scores, undoing the last event, or viewing the event log.
6.  **Lock/Unlock UI:** Toggle the UI lock from the main menu to prevent accidental input.
7.  **Exit Game:** Use the 'Back' button to bring up the exit menu, where you can choose to resume, end, reset, or save the game.
